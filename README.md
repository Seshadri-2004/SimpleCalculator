# SimpleCalculator

A basic Java command-line calculator supporting:

- Addition `+`
- Subtraction `-`
- Multiplication `*`
- Division `/`
- Modulus `%`
- Exponentiation `^`

## How to Run

```bash
javac SimpleCalculator.java
java SimpleCalculator
```

## Sample

```
Enter first number: 10
Enter operator (+, -, *, /, %, ^): *
Enter second number: 5
Result: 50.0
```
